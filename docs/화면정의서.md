## SCR-01. 로그인

### 기본 정보

| 항목 | 내용 |
|---|---|
| 화면 ID | SCR-01 |
| 화면명 | 로그인 |
| URL | `/login` |
| 접근 권한 | 비로그인 사용자 |

### 화면 레이아웃

```
┌──────────────────────────────────────┐
│                                      │
│         영업 일일보고 시스템          │
│                                      │
│   이메일  [________________________] │
│   비밀번호 [________________________] │
│                                      │
│           [    로그인    ]            │
│                                      │
└──────────────────────────────────────┘
```

### 입력 항목

| 항목 | 타입 | 필수 | 유효성 |
|---|---|---|---|
| 이메일 | text (email) | Y | 이메일 형식 |
| 비밀번호 | password | Y | 8자 이상 |

### 기능 버튼

| 버튼 | 동작 |
|---|---|
| 로그인 | 인증 성공 시 대시보드(SCR-02)로 이동 |

### 오류 메시지

| 조건 | 메시지 |
|---|---|
| 이메일 미입력 | 이메일을 입력하세요 |
| 비밀번호 미입력 | 비밀번호를 입력하세요 |
| 인증 실패 | 이메일 또는 비밀번호가 올바르지 않습니다 |
| 퇴직 계정 | 사용이 중지된 계정입니다 |

| 화면명 | 대시보드 |

### 화면 레이아웃

```
┌─────────────────────────────────────────────────────┐
│ 영업 일일보고 시스템     [사원명] [로그아웃]         │
├──────────┬──────────────────────────────────────────┤
│ 메뉴     │  오늘의 보고 현황                         │
│          │  ┌───────────┬───────────┬─────────────┐ │

│          │  │ 사원명   보고일자   상태             │ │

│          │  │ 김영희   2026-05-13  작성중          │ │
│          │  └─────────────────────────────────────┘ │
```

### 표시 항목

| 영역 | 내용 | 표시 조건 |
|---|---|---|
| 오늘의 보고 현황 | 금일 기준 상태별 건수 (작성중/제출/확인) | 상급자: 팀 전체, 사원: 본인 |
| 최근 미제출 보고 | 전일 기준 미작성·작성중 목록 | 상급자만 표시 |
| 오늘 보고서 작성하기 버튼 | 금일 보고서 없으면 작성(SCR-05), 있으면 상세(SCR-06)로 이동 | 영업사원만 표시 |

## SCR-03. 일일보고 목록 (본인)

### 기본 정보

| 항목 | 내용 |
|---|---|
| 화면 ID | SCR-03 |
| 화면명 | 일일보고 목록 (본인) |
| URL | `/report/my` |
| 접근 권한 | 로그인 사용자 전체 |

### 화면 레이아웃

```
┌──────────────────────────────────────────────────────────────┐
│ 나의 일일보고 목록                         [ + 보고서 작성 ] │
├──────────────────────────────────────────────────────────────┤
│ 조회기간: [2026-04-01] ~ [2026-05-14]   상태: [전체 ▼]  [조회] │
├──────────────────────────────────────────────────────────────┤
│  No │ 보고일자    │ 방문건수 │ 상태    │ 제출일시          │
│  1  │ 2026-05-14 │    3건   │ 작성중  │ -                 │
│  2  │ 2026-05-13 │    2건   │ 확인완료│ 2026-05-13 18:02  │
│  3  │ 2026-05-12 │    4건   │ 제출완료│ 2026-05-12 17:45  │
│ ... │            │          │         │                   │
├──────────────────────────────────────────────────────────────┤
│ << 1 2 3 ... >>                                              │
└──────────────────────────────────────────────────────────────┘
```

### 검색 조건

| 항목 | 타입 | 기본값 | 설명 |
|---|---|---|---|
| 조회기간 (시작) | date | 당월 1일 | |
| 조회기간 (종료) | date | 오늘 | |
| 상태 | select | 전체 | 전체 / 작성중 / 제출완료 / 확인완료 |

### 목록 컬럼

| 컬럼 | 설명 |
|---|---|
| 보고일자 | 클릭 시 일일보고 상세(SCR-06)로 이동 |
| 방문건수 | 해당 보고서의 visit_record 건수 |
| 상태 | DRAFT: 작성중, SUBMITTED: 제출완료, CONFIRMED: 확인완료 |
| 제출일시 | 미제출이면 `-` 표시 |

### 기능 버튼

| 버튼 | 동작 |
|---|---|
| + 보고서 작성 | SCR-05로 이동. 금일 보고서가 이미 존재하면 경고 후 해당 보고서로 이동 |
| 조회 | 검색 조건 적용하여 목록 재조회 |

## SCR-05. 일일보고 작성/수정

### 기본 정보

| 항목 | 내용 |
|---|---|
| 화면 ID | SCR-05 |
| 화면명 | 일일보고 작성/수정 |

| 접근 권한 | 영업사원 본인 보고서, DRAFT 상태일 때만 수정 가능 |

### 화면 레이아웃

```
┌──────────────────────────────────────────────────────────────┐
│ 일일보고 작성              보고일자: 2026-05-14  작성자: 홍길동 │
├──────────────────────────────────────────────────────────────┤
│ [방문기록]  [과제/상담]  [익일계획]                          │
├──────────────────────────────────────────────────────────────┤
│ ▶ 방문기록                                      [ + 행 추가 ] │
│ ┌──────┬──────────┬───────────┬──────────────┬────────────┐  │
│ │고객  │방문시각  │방문목적   │방문내용      │차기방문예정│  │
│ │[검색]│[HH:MM]  │[________] │[___________] │[날짜선택]  │  │
│ │[검색]│[HH:MM]  │[________] │[___________] │[날짜선택]  │  │
│ └──────┴──────────┴───────────┴──────────────┴────────────┘  │
├──────────────────────────────────────────────────────────────┤
│ ▶ 과제/상담 (Problem)                           [ + 행 추가 ] │
│ ┌────────────────────────────────────────────────────────┐   │
│ │ 1. [____________________________________________] [삭제] │   │
│ │ 2. [____________________________________________] [삭제] │   │
│ └────────────────────────────────────────────────────────┘   │
├──────────────────────────────────────────────────────────────┤
│ ▶ 익일계획 (Plan)                               [ + 행 추가 ] │
│ ┌────────────────────────────────────────────────────────┐   │
│ │ 1. [____________________________________________] [삭제] │   │
│ └────────────────────────────────────────────────────────┘   │
├──────────────────────────────────────────────────────────────┤
│                     [임시저장]      [제출]                    │
└──────────────────────────────────────────────────────────────┘
```

### 방문기록 입력 항목

| 항목 | 타입 | 필수 | 설명 |
|---|---|---|---|
| 고객 | 검색 팝업 | Y | 고객명·회사명으로 검색 (SCR-07 팝업) |
| 방문시각 | time (HH:MM) | N | |
| 방문목적 | text | N | 최대 500자 |
| 방문내용 | textarea | N | |
| 차기방문예정일 | date | N | |
| 삭제 버튼 | button | - | 해당 행 삭제 |

### 과제/상담 입력 항목

| 항목 | 타입 | 필수 | 설명 |
|---|---|---|---|
| 내용 | textarea | Y (행이 있을 경우) | |
| 삭제 버튼 | button | - | 해당 행 삭제 |

### 익일계획 입력 항목

| 항목 | 타입 | 필수 | 설명 |
|---|---|---|---|
| 내용 | textarea | Y (행이 있을 경우) | |
| 삭제 버튼 | button | - | 해당 행 삭제 |

### 기능 버튼

| 버튼 | 동작 | 조건 |
|---|---|---|
| + 행 추가 | 각 섹션에 입력 행 추가 | |
| 임시저장 | status = DRAFT 로 저장, 현재 화면 유지 | |
| 제출 | 유효성 검증 후 status = SUBMITTED 로 저장, SCR-06으로 이동 | 방문기록 1건 이상 필수 |

### 유효성 검증

| 항목 | 규칙 |
|---|---|
| 방문기록 | 제출 시 1행 이상 존재해야 함 |
| 고객 | 방문기록 행이 있을 경우 고객 선택 필수 |
| 과제/상담 내용 | 행이 추가된 경우 내용 필수 |
| 익일계획 내용 | 행이 추가된 경우 내용 필수 |
| 수정 가능 상태 | DRAFT 상태일 때만 접근 허용 |

### 표시 항목

| 버튼 | 동작 | 표시 조건 |

## 공통 사항

### 권한 정의

| 역할 | 정의 | 구분 기준 |
|---|---|---|
| 영업사원 | 일반 사용자 | `manager_id`가 NULL이 아닌 사원 |
| 상급자 | 부하직원이 존재하는 사원 | 타 사원의 `manager_id`로 참조되는 사원 |
| 관리자 | 마스터 관리 권한 | 별도 플래그 또는 최상위 직급 |

| 중복 | 이미 사용 중인 {항목명}입니다 |
